# 隐私说明

## 一、原则

Mango 的隐私原则是：本地优先、最小采集、明确告知、用户可控。

## 二、当前本地行为

当前版本默认将以下内容保存在本地：

- 桌面设置
- 活跃任务
- 历史任务
- 工作区相关状态

## 三、当前不会做的事

当前版本默认不会：

- 上传原始提示词全文
- 上传文件内容
- 上传环境变量
- 上传密钥

## 四、未来如果接入真实远端能力

如果后续接入真实 CLI 或远端 Provider，必须满足：

- 在文档中明确说明会访问哪些外部服务
- 在 UI 中明确告诉用户
- 网络能力保持可见与可控

## 五、用户控制能力

后续正式版本需要保证用户可以：

- 开关遥测
- 选择发布通道
- 查看当前使用的外部能力
